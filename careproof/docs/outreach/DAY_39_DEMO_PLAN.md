## Demo Flow To Use If A Call Is Booked

1. Discovery:
   - How do families know a visit was completed?
   - How often do families call asking for updates?
   - How do caregivers document visits today?
2. Positioning:
   - CareProof is not replacing scheduling or billing.
   - It focuses on family updates, visit proof, and caregiver documentation.
3. Show:
   - caregiver workflow
   - family visit update
   - agency proof timeline
   - 14-day small pilot option
4. Close:
   - Would you test this with 2–3 caregivers and 3–5 clients for 14 days?
