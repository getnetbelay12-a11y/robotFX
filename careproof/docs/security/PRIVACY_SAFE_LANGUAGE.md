# Privacy-Safe Language

## Use

“Built with role-based access, audit logs, secure authentication, and agency-level data isolation.”

“CareProof supports operational communication and proof-of-care workflows.”

“CareProof is not an emergency service. For emergencies, call 911.”

## Avoid

- “HIPAA compliant”
- “medical diagnosis”
- “clinical risk scoring”
- “emergency response”
- “guaranteed fraud prevention”

Do not overclaim compliance or clinical scope before legal, infrastructure, and workflow review are complete.
