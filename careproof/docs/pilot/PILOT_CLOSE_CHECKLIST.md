# CareProof Pilot Close Checklist

- [ ] Agency owner agrees to pilot
- [ ] One admin champion assigned
- [ ] At least 5 caregivers will test
- [ ] At least 5 clients included
- [ ] Family updates allowed
- [ ] Weekly review meeting agreed
- [ ] Success metrics agreed
- [ ] Start date confirmed
- [ ] Data import template completed

No champion means the pilot is likely to fail.
