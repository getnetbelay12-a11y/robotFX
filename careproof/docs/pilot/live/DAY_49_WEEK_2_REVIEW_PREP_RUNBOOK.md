## Key Percentages

Calculate:

- visit start rate = visits started / scheduled visits
- visit completion rate = visits completed / scheduled visits
- checklist completion rate = completed visits with checklist / completed visits
- care note completion rate = completed visits with care note / completed visits
- family update rate = family updates generated / completed visits

## Honest Interpretation

Strong:

- 80%+ visit start rate
- 85%+ checklist completion
- 85%+ care note completion
- 80%+ family update generation

Mixed:

- 50-79% visit start rate
- documentation inconsistent
- family updates working but not consistently

Weak:

- below 50% visit start rate
- caregivers not adopting
- admin not using dashboard
- family updates not creating value

Do not sugarcoat weak metrics. Weak metrics mean fix the operating workflow or reduce the pilot group.

## Value Signal Review

Strong:

- agency says family update calls decreased
- admin caught late or missed visit earlier
- caregiver documentation improved
- family concerns are easier to track
- weekly report is useful
- agency would miss CareProof if removed

Weak:

- families still require manual texts
- admin does not check dashboard
- caregivers only use app when pushed
- reports are ignored
- nobody can name what CareProof made easier

If the agency cannot name value, you are not ready to sell. You are ready to diagnose.

## Conversion-Signal Questions

Ask the champion:

- What has CareProof made easier so far?
- What still feels like extra work?
- Did family update calls or texts decrease?
- Did caregiver documentation improve?
- Did the dashboard help you catch anything earlier?
- Would your team miss CareProof if we stopped after the pilot?
- Who should join the next review if we discuss continuation?

Most important:

If the pilot continues to show these results, would the Starter plan at $149/month be reasonable?

## Recommendation Options

- Prepare paid continuation
- Continue with targeted fixes
- Reduce scope
- Stop or poor fit

False optimism wastes time.
