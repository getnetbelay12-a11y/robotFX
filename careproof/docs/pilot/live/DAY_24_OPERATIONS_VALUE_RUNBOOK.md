# Day 24 Operations Value Runbook

## Goal

Determine whether CareProof is helping the agency run better, not just document visits.

Family updates help sell the product, but agencies pay when they feel:

- We have better control
- We catch issues earlier
- We spend less time chasing updates
- We know which caregivers and clients need attention

## Review Dashboard Usage

Ask the pilot champion:

1. How often did you check the dashboard this week?
2. Which dashboard cards were useful?
3. Which cards were ignored?
4. Did late and missed visits help?
5. Did the incident queue help?
6. Did the family concern queue help?
7. Did caregiver reliability help?
8. What did you still track manually?

If the admin is not using the dashboard, agency value is weak.

## Review Operational Value Areas

### Visit Operations

- scheduled visits
- completed visits
- late visits detected
- missed visits detected
- requires-review visits
- visits corrected by admin

### Incident Management

- incidents submitted
- incidents reviewed
- incidents resolved
- average review time
- open incidents

### Family Concern Management

- concerns submitted
- concerns reviewed
- concerns resolved
- average response time
- open concerns

### Caregiver Reliability

Classify:

- Reliable
- Needs coaching
- Remove from pilot

## Strong Value Signal

CareProof showed us a late or missed visit before the family called.

That is operational value the agency can feel.
