## Key Rates

Use percentages, not just counts.

- visit start rate = visits started / scheduled visits
- visit completion rate = visits completed / scheduled visits
- checklist completion rate = completed visits with checklist / completed visits
- care note completion rate = completed visits with care note / completed visits
- family update rate = family updates generated / completed visits

## Honest Interpretation

Strong:

- 80%+ start rate
- 85%+ checklist completion
- 85%+ care note completion
- 80%+ family update generation

Mixed:

- 50-79% start rate
- inconsistent checklist or note completion
- caregivers still need reminders

Weak:

- below 50% start rate
- admin not using dashboard
- family updates not working
- caregivers resisting

Do not call weak numbers promising. Weak numbers mean intervention.

## Caregiver Classification

Reliable:

- starts visits
- completes checklist
- adds notes
- checks out

Needs coaching:

- uses app but misses one repeated step

Pause:

- does not start visits
- repeatedly cannot complete workflow
- creates confusion for the pilot

## Family Update Review

Check:

- correct family recipient
- correct client
- clear visit summary
- no raw incident details
- no internal notes
- concern button works

## Admin Operations Review

Ask:

- Did you check the dashboard each day?
- Did late or missed visibility help?
- Did the proof timeline help answer any question?
- Did the incident or family concern queue help?
- What are you still tracking outside CareProof?
