# Caregiver Coaching Message

Hi [Caregiver Name],

Thank you for helping with the CareProof pilot.

For each assigned visit, please remember this simple flow:

1. Open CareProof
2. Tap Today
3. Start the visit
4. Complete the checklist
5. Add a short care note
6. Report an issue if needed
7. End the visit

The most important steps are Start Visit, Checklist, Care Note, and End Visit.

This helps the agency confirm care was delivered and helps families receive clear updates.

If your visit does not show or you have trouble logging in, please contact [Pilot Champion Name] right away.
