## Path A: Continue Same Group

Use if Day 46 numbers were strong.

Focus:

- maintain consistency
- do not expand yet
- confirm checklist, note, and family update completion
- confirm admin checked dashboard

Target:

- 80%+ visits started
- 85%+ checklist completion
- 85%+ care note completion
- 80%+ family update generation

## Path B: Continue With Coaching

Use if workflow works, but some caregivers miss steps.

Focus:

- identify exact repeated mistake
- send reminder
- do 1:1 coaching if needed
- recheck same caregiver on next shift

## Path C: Reduce Pilot Group

Use if adoption is weak.

Focus:

- keep only most reliable caregivers and clients
- remove confusion
- prove workflow cleanly in smaller group

This is not failure. It is damage control.

## Path D: Pause and Fix

Use only if there are serious blockers.

Pause if:

- wrong client or family data
- unsafe family update
- caregiver cannot complete visit
- checkout fails
- proof timeline broken
- multiple logins broken

Do not relaunch until a controlled test passes again.

## Core Workflow Recheck

Regardless of path, verify again:

1. Admin verifies visit
2. Caregiver sees assigned visit
3. Caregiver starts visit
4. Checklist saves
5. Care note saves
6. Checkout works
7. Family update generated
8. Admin proof timeline shows events
9. Family sees safe update only

If this fails, do not expand the pilot.

## Family Update Safety

Safe:

Today’s visit was completed. Meal assistance, hydration support, and companionship were completed. The caregiver noted: No major concerns.

Review-safe:

Today’s visit was completed and is being reviewed by the agency. The agency will follow up if needed.

Unsafe:

High-severity incident: fall in bathroom...

Unsafe update means pause and fix.
