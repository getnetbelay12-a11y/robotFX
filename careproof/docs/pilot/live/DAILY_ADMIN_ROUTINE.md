# Daily Admin Routine — CareProof Pilot

## Morning

- [ ] Review today's scheduled visits
- [ ] Confirm pilot caregivers have assigned visits
- [ ] Check for missing assignments
- [ ] Remind caregivers of the Start -> Checklist -> Note -> End flow

## During the Day

- [ ] Watch late visits
- [ ] Watch missed check-ins
- [ ] Review incidents
- [ ] Review family concerns

## End of Day

- [ ] Review completed visits
- [ ] Check incomplete checklists
- [ ] Check missing care notes
- [ ] Confirm family updates were generated
- [ ] Follow up with caregivers who missed steps
- [ ] Log issues for weekly review

## Why This Matters

Without this routine, the pilot becomes passive and adoption falls quickly.
