# Day 24 Late and Missed Visit Value Check

## Questions

1. How many late visits were detected?
2. How many missed visits were detected?
3. Did the agency act faster because of CareProof?
4. Did the dashboard catch anything the team would have missed?
5. Did any family complaint get prevented?

## Best Signal

CareProof showed us a late or missed visit before the family called.

## If Value Is Weak

If the agency says the late and missed view is not useful, find out why:

- counts are wrong
- champion is not checking dashboard
- late definition does not match agency expectations
- coordinators still rely on another system first
