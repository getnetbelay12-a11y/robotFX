# Day 40 Pilot Setup Runbook

## Goal

Turn demo interest into a real pilot setup with:

- named agency
- named pilot champion
- selected caregivers
- selected clients
- family update method
- target start date
- agreed success metrics

If those are not confirmed, the pilot is not real yet.

## 30-Minute Setup Call

### 0-5 Minutes: Confirm Pilot Goal

Say:

The goal today is to keep the pilot small and measurable.

We are not rolling this out across the whole agency. We are testing whether CareProof can reduce manual family updates, improve caregiver documentation, and give the agency a clear proof-of-care timeline.

### 5-10 Minutes: Confirm Pilot Champion

Ask directly:

- Who will own the pilot day-to-day?
- Who will remind caregivers?
- Who will review the dashboard?
- Who will handle family concerns?

No champion means no pilot.

### 10-15 Minutes: Confirm Scope

Default scope:

- duration: 14 days
- caregivers: 2-3
- clients: 3-5
- family members: selected by agency
- visit types: normal non-emergency care visits

If they ask for more:

We should start smaller first. If the first group works, expansion is easy. If the first group struggles, a larger rollout will create confusion and bad data.

### 15-20 Minutes: Confirm Data Import

Confirm:

- who sends `caregivers.csv`
- who sends `clients.csv`
- who sends `family_members.csv`
- who sends `visits.csv`
- when the files will be sent

Warn them:

For the first pilot, please do not include unnecessary sensitive medical information. Keep the data limited to what is needed to run the visit workflow and family updates.

### 20-25 Minutes: Confirm Success Metrics

Use:

- 80%+ pilot visits completed through CareProof
- 85%+ completed visits include checklist
- 85%+ completed visits include care note
- 80%+ completed visits generate family update
- no unsafe family updates
- agency confirms whether manual family update calls decreased

Ask:

If the pilot hits these numbers and reduces manual family updates, would you consider continuing on the Starter plan?

### 25-30 Minutes: Confirm Start Date and Next Actions

Lock:

- pilot champion
- start date
- CSV owner
- training date
- weekly review date
- CareProof import owner

## Non-Negotiables

- no champion
- no pilot
- no selected caregivers/clients
- no pilot
- no target start date
- no pilot
- no success metrics
- no pilot
