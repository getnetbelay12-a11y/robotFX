# Day 22 Execution Runbook

## Goal

Make CareProof a repeated daily habit for caregivers and admins.

Week 2 is no longer about whether the product can work once. It is about whether the workflow can hold up daily with less confusion and less support friction.

## Start With The Week 2 Plan

Open:

`docs/pilot/live/[agency_name]/WEEK_2_PLAN.md`

Confirm which path was chosen:

- Continue same group
- Continue with coaching
- Reduce pilot group
- Pause and fix

Your Day 22 actions depend on that decision.

## If Week 2 = Continue Same Group

Focus on consistency.

Check:

- Are caregivers starting visits on time
- Are checklists being completed
- Are notes being added
- Are family updates generating
- Is the admin champion checking the dashboard daily

Target:

- 80%+ visits started in CareProof
- 85%+ completed visits with checklist
- 85%+ completed visits with care note
- 80%+ family updates generated

## If Week 2 = Continue With Coaching

Focus on weak caregivers. Coaching is the main intervention, not product expansion.

## If Week 2 = Reduce Pilot Group

Reduce to:

- 2 to 3 reliable caregivers
- 3 to 5 clients
- family updates enabled
- daily admin monitoring

Do not expand until the smaller group works reliably.

## If Week 2 = Pause And Fix

Pause only for serious blockers:

- wrong-family or wrong-client issue
- caregiver cannot complete visits
- family updates are unsafe
- admin dashboard unusable
- login broken

Do not continue a broken pilot just to look busy.

## Priority Fix Categories

1. Login and access
2. Visit visibility
3. Check-in and check-out
4. Checklist save
5. Care note save
6. Family update generation
7. Dashboard metric accuracy
8. Incident and family concern visibility
