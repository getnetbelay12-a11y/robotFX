# Day 24 Agency Value Questions

Ask the agency directly:

1. What did CareProof make easier this week?
2. What did CareProof make harder?
3. Did it save your coordinator any time?
4. Did it reduce family update calls or texts?
5. Did it help you catch any issue earlier?
6. Would you miss it if we turned it off tomorrow?

## Scoring Guidance

### Strong agency value

- Admin checks dashboard daily
- Late and missed visits are visible
- Incidents and concerns are tracked in system
- Caregiver issues are identified
- Family update calls are reduced
- Agency wants to continue

### Mixed agency value

- Dashboard useful sometimes
- Caregiver adoption uneven
- Family updates useful
- Admin still tracks many things manually
- Agency wants more refinement

### Weak agency value

- Admin rarely checks dashboard
- Caregivers do not use app consistently
- Family updates not used
- Incidents still handled outside the system
- No clear time saving
