# Family Concern Flow Test

Run one real or controlled test and verify:

- [ ] Family submits concern
- [ ] Admin sees concern in queue
- [ ] Admin assigns or reviews concern
- [ ] Admin responds or marks status
- [ ] Concern is linked to the correct client
- [ ] No unrelated family or client data is visible

## Suggested Concern Categories

- Care quality
- Schedule
- Caregiver
- Health concern
- Billing or admin
- Other

## Rule

If the concern flow is hard, families will still call instead of using the product.
