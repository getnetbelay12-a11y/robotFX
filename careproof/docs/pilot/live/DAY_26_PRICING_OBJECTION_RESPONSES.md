# Day 26 Pricing Objection Responses

## Can we keep testing for free?

We can extend the pilot only if there are specific success criteria that were not fully tested yet. Otherwise, the pilot has already shown whether the workflow is useful. If you want to continue using it operationally, it should move to the Starter plan.

## $149 per month is too much

Understood. The question is whether CareProof saves enough time, improves family communication, and protects client trust enough to justify less than the cost of one small administrative mistake or a few hours of coordinator time.

Then ask:

What price would feel reasonable based on the value you saw?

## We need more features first

Which feature is required to continue, and which feature is nice to have?

Then sort each request into:

- Blocker
- Important
- Nice-to-have

Do not build every request.

## We need EVV

CareProof is not replacing formal EVV in this version. If EVV is required for all of your workflows, we should either limit CareProof to private-pay and family communication workflows or pause until EVV integration becomes part of the roadmap.

## Caregivers are still inconsistent

That means the question is whether the issue is product friction, training, or caregiver discipline. If the workflow works for some caregivers but not others, the extension should focus on coaching and adoption, not new features.
