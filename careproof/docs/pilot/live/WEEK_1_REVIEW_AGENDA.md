# Week 1 Review Agenda

## Goal

Decide whether CareProof is helping the agency reduce family communication burden and improve caregiver accountability.

## 30-Minute Structure

### 0-5 Minutes: Pilot Goal Reminder

Start with:

The purpose of this review is to decide whether CareProof is helping your team reduce family communication burden and improve caregiver accountability.

### 5-10 Minutes: Usage Metrics

Review:

- scheduled visits
- started visits
- completed visits
- late and missed visits
- requires-review visits

### 10-15 Minutes: Caregiver Adoption

Review:

- good adopters
- low-usage caregivers
- coaching needs
- workflow friction

### 15-20 Minutes: Family Updates and Concerns

Review:

- updates generated
- updates understood
- concerns submitted
- wording issues

### 20-25 Minutes: Issues and Fixes

Review:

- high-priority issues
- fixes applied
- remaining risks

### 25-30 Minutes: Week 2 Decision

Choose:

- continue same group
- continue with coaching
- reduce pilot group

## Mandatory Questions

1. Did CareProof reduce any family update calls this week?
2. Did the dashboard help your team see what was happening?
3. Which caregivers adapted fastest?
4. Which step caused the most confusion?
5. Were family updates useful and safe?
6. Would you want more clients included next week, or should we stabilize first?
7. What would need to improve before you would pay for this?
