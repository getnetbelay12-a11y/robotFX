# Day 28 Execution Paths

Only three paths are valid after the pilot decision:

## Path A — Paid Customer

Execute paid onboarding.

- confirm billing contact
- confirm paid start date
- confirm active user and client scope
- send invoice or payment setup
- document first paid month goals

## Path B — Controlled Extension

Execute a short extension only.

- confirm extension dates
- confirm success criteria
- keep scope fixed
- schedule final decision meeting

## Path C — Lost Pilot

Execute clean closeout.

- document loss reason
- extract product lessons
- adjust targeting and sales message
- prioritize next prospects

## Rule

Do not mix these paths. Pick the real one and execute it.
