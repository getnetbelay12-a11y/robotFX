# Day 26 Missing Proof Before Conversion — [Agency Name]

| Missing Proof | Why It Matters | Action |
|---|---|---|
| Family call reduction not measured | Core value claim | Ask admin to compare call volume |
| Low caregiver note completion | Weak documentation value | Coach caregivers |
| Dashboard not checked daily | Weak ops value | Enforce daily routine |
| Decision-maker not involved | Cannot close | Invite owner or admin |

## Rule

If you do not know what proof is missing, you are not ready to close.
