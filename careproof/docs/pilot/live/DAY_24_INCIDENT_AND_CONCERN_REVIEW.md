# Day 24 Incident and Concern Review

## Incident Workflow Questions

1. Were incidents submitted through CareProof?
2. Did admin see them quickly?
3. Were incidents assigned or reviewed?
4. Were sensitive incidents kept family-safe?
5. Did the agency prefer this over text or calls?

## If Incidents Still Happen Outside The System

Ask why:

- Was the incident form too long?
- Did the caregiver forget?
- Was severity confusing?
- Did admin not reinforce the workflow?

## Family Concern Workflow Questions

1. Did family concerns arrive in the queue?
2. Did admin respond?
3. Were concerns easier to track than calls or texts?
4. Were any concerns unresolved too long?

## Target Feeling

Now we have one place to track family concerns.
