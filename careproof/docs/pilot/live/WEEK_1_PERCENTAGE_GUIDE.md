# Week 1 Percentage Guide

Use these formulas to interpret pilot performance honestly.

## Formulas

Visit adoption rate =
`visits started in CareProof / scheduled pilot visits`

Visit completion rate =
`visits completed in CareProof / scheduled pilot visits`

Checklist completion rate =
`completed visits with checklist / completed visits`

Care note completion rate =
`completed visits with care note / completed visits`

Family update rate =
`family updates generated / completed visits`

## Example

- Scheduled pilot visits: 40
- Started in CareProof: 30 -> 75%
- Completed in CareProof: 28 -> 70%
- Checklist completed: 26 / 28 -> 93%
- Care note added: 24 / 28 -> 86%
- Family updates generated: 25 / 28 -> 89%

## Interpretation

### Strong pilot signal

- 75%+ visits started
- 80%+ completed visits include checklist
- 80%+ completed visits include care note
- 80%+ family updates generated
- No critical safety issue
- Champion wants to continue

### Mixed signal

- 50% to 74% visits started
- Checklist or note completion inconsistent
- Caregivers need reminders
- Family updates work but not always
- Champion sees value but adoption needs work

### Weak signal

- Below 50% visits started
- Caregivers resist
- Family updates not used
- Admin not checking dashboard
- Champion inactive
- Major workflow confusion
