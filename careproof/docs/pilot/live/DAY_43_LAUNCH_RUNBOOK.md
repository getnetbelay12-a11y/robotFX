# Day 43 New Agency Pilot Launch

## Goal

Launch the pilot with real users and verify that:

- caregivers can complete real visits safely
- families receive useful updates safely
- admin can verify the proof timeline

Today is not feature day. It is real-user execution day.

## Morning Launch Checklist

- [ ] Admin or champion can log in
- [ ] Selected caregivers can log in
- [ ] Today's visits are visible
- [ ] Caregivers see only assigned visits
- [ ] Family members are linked correctly
- [ ] Family communication method is approved
- [ ] Support contact is confirmed
- [ ] Day 42 test visit passed
- [ ] No critical or high issue remains open

If any login or wrong-data issue appears, pause launch.

## Caregiver Reminder

Hi [Caregiver Name],

Reminder: today we are starting the CareProof pilot.

For every assigned visit, please follow this simple flow:

1. Open CareProof
2. Tap Today
3. Open your visit
4. Tap Start Visit
5. Complete the checklist
6. Add a short care note
7. Report an issue if needed
8. Tap End Visit

If your visit does not show or you cannot log in, contact [Pilot Champion Name] right away.

## Live Monitoring Priority

1. Data safety
2. Login and access
3. Visit visibility
4. Check-in and checkout
5. Checklist and note
6. Family update
7. Dashboard and reporting
8. UI polish

Do not fix cosmetic issues while visits are blocked.

## End-of-Day Champion Review

Ask:

- Which caregivers used it successfully?
- Where did caregivers get stuck?
- Did visits show correctly?
- Did check-in and checkout work?
- Were checklists easy enough?
- Were care notes completed?
- Were family updates useful?
- Did dashboard and proof timeline help?
- What must be fixed before tomorrow?

Then ask directly:

Are we good to continue tomorrow with the same pilot group?
