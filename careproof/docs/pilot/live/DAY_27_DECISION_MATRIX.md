# Day 27 Decision Matrix

## Continue Paid

Use when:

- the decision-maker sees visible value
- starter pricing is acceptable
- the team wants to keep using CareProof operationally

## Extend With Clear Criteria

Use when:

- the agency sees potential value
- one or two proof points are still missing
- there is a named decision-maker and a defined 7-day window

## Stop

Use when:

- value is not visible
- the wrong customer fit is obvious
- adoption is too weak
- required features are outside current scope

## Weak Outcome To Reject

“Let’s keep talking” with no decision-maker, no metrics, and no date is not progress.
