# Caregiver Quick Start

Caregiver training should take less than 10 minutes.

## Workflow

1. Open the app
2. Login
3. Tap today’s visit
4. Tap Start Visit
5. Complete checklist
6. Add note
7. Report issue if needed
8. Tap End Visit

## Key reminder

The app is designed to keep documentation simple. If the workflow feels slow or confusing, the agency should report that immediately.
