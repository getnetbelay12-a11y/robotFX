# CareProof Day 81-90 Plan

## Selected Focus

- [ ] A — Customer Success + Proof
- [ ] B — Conversion
- [ ] C — New Pilot Creation
- [x] D — Outreach / Discovery

## Main Goal

Create the first real market conversations and prove whether the message, segment, and channel can generate qualified discovery calls.

## Top 3 Metrics

1. Emails sent
2. Calls made
3. Conversations started

## Day 81

Send 10 warm follow-up emails, make 5 high-fit calls with the call-first opener, and update every touched tracker row the same day.

## Day 82

Add 20 qualified agencies, send 10 more emails, make 5 more calls, and compare first live results by message angle.

## Day 83

Follow up on every reply within 24 hours, push for 10-minute discovery calls, and close or deprioritize dead rows.

## Day 84

Double down only on segments or messages that show live signal. If there is still zero signal, increase phone-heavy outreach and close-loop messaging.

## Day 85 Review

Review live outreach numbers, identify the first real winning or losing pattern, and decide whether message, segment, or channel needs to change.

## Day 86

Run the winning outreach pattern again or, if there is still zero signal, change the outreach channel and reduce time spent on non-responsive rows.

## Day 87

Follow up on all live responses, push for short discovery calls, and remove stale rows from active focus.

## Day 88

Expand only the segment that shows response. If none respond, test a new segment with the same call-first discipline.

## Day 89

Prepare a clean movement summary: conversations, demos, objections, and closures.

## Day 90 Review

Judge whether CareProof now has real sales signal or whether the business is still blocked by non-execution.

## Do Not Build

- Billing
- Payroll
- EVV
- AI diagnosis
- Custom branding
- Advanced dashboards
- Chat/video
- Marketplace

## Success Criteria

- first live outreach execution logged
- first real conversation started
- touched-row statuses updated consistently
- weak opportunities closed instead of lingering

## Failure Criteria

- no live outreach execution logged
- no touched-row tracker updates
- no conversation or objection data created
- stalled opportunities still counted as active pipeline
