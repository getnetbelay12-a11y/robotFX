# CareProof Operating Playbook

## Current Best Customer Profile

CareProof currently fits best with:

- private-pay non-medical home care agencies
- 10–50 caregivers first
- owner-operated agencies or small admin teams
- agencies where family update calls are common
- agencies with inconsistent caregiver documentation
- agencies that do not want to replace scheduling or billing software
- agencies that can assign one pilot champion

### Poor Fit

Avoid first:

- agencies requiring EVV before anything else
- agencies asking for payroll or billing as the first priority
- large enterprise or highly layered providers
- agencies with no visible family communication pain
- agencies unwilling to assign a pilot champion

## Best Sales Message

Current default message:

CareProof helps home care agencies reduce family update calls by sending clear visit summaries after caregiver checkout while keeping a proof-of-care timeline for the agency.

Short version:

CareProof helps agencies prove visits and update families without replacing their current scheduling or billing system.

### Important note

This is the current default message, not a proven winner yet. The Month 2 outreach test still needs to validate whether:

- family updates
- caregiver accountability
- owner visibility / proof-of-care

gets the strongest reply rate.

## Best Demo Flow

### 1. Discovery — 3 minutes

Ask:

- How do families know a visit was completed?
- How do caregivers document visits today?
- How often do families call asking for updates?

### 2. Positioning — 2 minutes

CareProof does not replace scheduling or billing.

It focuses on:

- visit proof
- family updates
- incident visibility
- weekly reporting

### 3. Product Demo — 10 minutes

Show:

- admin dashboard
- visit proof timeline
- caregiver workflow
- family update
- incident / concern queue
- weekly report

### 4. Pilot Offer — 3 minutes

Offer the small pilot:

- 2–3 caregivers
- 3–5 clients
- 14 days
- family updates enabled
- success metric: fewer manual family updates and better documentation

### 5. Close — 2 minutes

Ask:

Would this be worth testing with a small group?

## Best Pilot Offer

### Recommended Pilot Offer

#### 14-Day Family Update + Proof-of-Care Pilot

Scope:

- 2–3 caregivers
- 3–5 clients
- selected family members
- caregiver check-in / check-out
- checklist
- care note
- family update
- incident / concern tracking
- weekly summary

Success criteria:

- 80%+ pilot visits completed through CareProof
- 85%+ checklist completion
- 85%+ care note completion
- 80%+ family updates generated
- agency confirms whether family calls or manual updates reduced

Why 14 days:

30 days is too much friction when buyer confidence is still low. A smaller pilot lowers commitment and forces a faster decision.

## Pilot Qualification Rules

### Pilot Qualification Checklist

- [ ] decision-maker or operations lead involved
- [ ] pilot champion assigned
- [ ] 2–3 caregivers selected
- [ ] 3–5 clients selected
- [ ] family updates allowed
- [ ] weekly review agreed
- [ ] success metrics agreed
- [ ] paid continuation discussed

No champion means:

- no accountability
- no usage discipline
- no clean conversion

## Pilot Setup Process

1. Confirm decision-maker and pilot champion.
2. Confirm 2–3 caregivers and 3–5 clients.
3. Confirm family update method.
4. Load caregiver, client, family, and visit data.
5. Train the champion first, then caregivers.
6. Run the first live visits with daily monitoring.
7. Review metrics weekly and force a continue / extend / stop decision.

## Pilot Success Metrics

Track:

- visit completion through CareProof
- checklist completion
- care note completion
- family update generation
- incidents and family concerns tracked
- agency-reported reduction in manual family update work

## Paid Conversion Process

1. Review pilot results with the decision-maker in the room.
2. Show before / after workflow change with real numbers.
3. Ask directly whether the Starter plan at $149/month makes sense.
4. If yes, move to paid setup.
5. If maybe, offer only a short extension with explicit criteria.
6. If no, record the loss reason cleanly and move on.

## Customer Success Process

### Daily during pilot

- check visit completion
- check checklist completion
- check care note completion
- check family updates
- check incidents and concerns
- send a quick issue summary if something is blocked

### Weekly

- send usage report
- review caregiver adoption
- review family communication quality
- review issues and fixes
- agree next actions

### First paid month

- weekly check-in
- adoption monitoring
- no uncontrolled expansion
- testimonial only after stable usage and visible value

## Best Customer Success Rhythm

Current best rhythm:

- daily monitoring during pilot
- one weekly review meeting
- one named champion
- one short issue summary when blockers happen

This is the current best rhythm because it is simple enough to repeat. It is not yet backed by multiple live customers.

## Hard Truth

This playbook is a working operating manual, not proof that the market already agrees. Anything marked “best” here is only as strong as the latest real customer or prospect behavior. If live evidence contradicts it, the playbook changes.
