# Day 36 Process Scorecard

## Sales Process

- Best target: private-pay non-medical home care agencies with visible family-trust messaging
- Best message: family updates as the current default angle
- Best CTA: small 14-day pilot or 10-minute conversation, not a broad platform pitch
- Best objection response: CareProof does not replace scheduling or billing; it focuses on family updates and visit proof
- Weakest part: no live outreach execution data yet

## Demo Process

- Best demo section: visit proof timeline plus family-safe update
- Weakest demo section: close confidence is still weak without real pilot proof
- Common objection: assumed “we already have software” defense, though still not measured live enough
- Improvement: run a shorter 15–20 minute problem-first demo with a smaller pilot offer

## Pilot Process

- Best scope: 14-day pilot with 2–3 caregivers and 3–5 clients
- Required champion: one agency owner, admin, or coordinator who owns the pilot daily
- Required success metrics: visit completion, checklist completion, care note completion, family update generation
- Common blocker: missing champion or weak buyer commitment

## Customer Success

- Best daily metric: completed visits with checklist and care note
- Best weekly metric: family updates generated plus agency feedback on manual update burden
- Biggest retention risk: adoption drift if the champion is passive

## Decision

- [x] Scale outreach
- [ ] Improve demo
- [ ] Start/continue pilot
- [ ] Close paid continuation
- [ ] Reposition

## Why This Decision Is Correct

The repo is demo-ready, but the business is still pre-demo in real terms. That means the next repeatable machine to build is outreach execution, not pilot operations or retention mechanics.

## Day 37 Execution Path

1. Add 20 more high-fit agencies so the current batch reaches 30.
2. Execute the three-message outreach test across the 30-agency batch.
3. Attempt 15 calls and log every outcome by segment and message angle.
4. Kill any message angle that still produces no movement after the first real batch.

## Hard Truth

The process is more repeatable now, but it is still not validated. The biggest weakness is not the product flow. It is the lack of enough real buyer behavior to confirm which message and offer truly win.
