# CareProof Sales Materials

Use these materials in this order:

1. Send `ONE_PAGE_CAREPROOF.md` before the first serious conversation.
2. Use `PITCH_DECK_OUTLINE.md` to build an 8–10 slide pitch.
3. Record or rehearse with `DEMO_SCRIPT.md`.
4. Send `PILOT_PROPOSAL.md` after a qualified demo.
5. Use `PRICING_SHEET.md` and `OBJECTION_HANDLING.md` during follow-up.

## Positioning Rule

CareProof is not a replacement for scheduling or billing software.

The message is:

CareProof helps home care agencies prove every visit, update every family, and catch care issues early.

## Compliance Wording

Do not say CareProof is HIPAA-compliant unless legal and infrastructure review are complete.

Use this instead:

Built with role-based access, audit logs, secure authentication, and agency-level data isolation.
