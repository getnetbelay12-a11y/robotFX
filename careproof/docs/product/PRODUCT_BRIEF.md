# CareProof Product Brief

## Product Promise

CareProof helps home care agencies prove every visit, update every family, reduce complaints, and catch care issues early.

## Target Customer

Small private-pay non-medical home care agencies with 10-100 caregivers.

## Core Users

- Agency owner/admin
- Care coordinator
- Caregiver
- Family member

## MVP Workflow

Agency creates visit
-> caregiver checks in
-> caregiver completes checklist
-> caregiver adds note
-> caregiver reports incident if needed
-> caregiver checks out
-> family receives update
-> agency sees proof timeline
-> weekly report generated

## MVP Modules

- Auth
- Agencies
- Users and roles
- Clients
- Care plans
- Visits
- Visit events
- Incidents
- Family concerns
- Dashboard
- Reports placeholder
- Notifications placeholder
- Audit logs

## Not Included in MVP

- Payroll
- Billing
- Medicaid EVV
- EHR integration
- Insurance claims
- AI diagnosis
- Caregiver marketplace
